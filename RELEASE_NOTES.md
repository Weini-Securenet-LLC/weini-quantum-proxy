## 🚀 Quick Start

### Desktop Application

**Windows**:
```bash
1. Download: weini-quantum-proxy-windows-amd64.exe
2. Run the application
3. Select a node
4. Click "Start Global Proxy"
```

**macOS**:
```bash
1. Download: weini-quantum-proxy-darwin-arm64 (M1/M2/M3)
   or: weini-quantum-proxy-darwin-amd64 (Intel)
2. Right-click → Open (first time only)
3. Select a node and connect
```

**Linux**:
```bash
1. Download: weini-quantum-proxy-linux-amd64
2. chmod +x weini-quantum-proxy-linux-amd64
3. ./weini-quantum-proxy-linux-amd64
```

### Mobile Devices

- **Android**: See [Android Guide](https://github.com/Weini-Securenet-LLC/weini-quantum-proxy/blob/main/docs/MOBILE_ANDROID.md)
- **iOS**: See [iOS Guide](https://github.com/Weini-Securenet-LLC/weini-quantum-proxy/blob/main/docs/MOBILE_IOS.md)
