## 🚀 快速开始

### 方式一：使用维尼量子节点获取配置（推荐）

#### 步骤1: 在电脑上运行维尼量子节点

1. 下载并安装维尼量子节点桌面版
2. 启动程序，获取可用节点
3. 选择一个或多个节点

#### 步骤2: 导出节点配置

**方法A: 二维码分享（最简单）**

1. 在维尼量子节点中选择节点
2. 点击"生成分享二维码"
3. 手机扫描二维码
4. 自动导入到客户端

**方法B: 链接分享**

1. 在维尼量子节点中复制节点链接
2. 通过聊天软件发送到手机
3. 在手机上打开链接
4. 选择用客户端打开

**方法C: 订阅链接（批量导入）**

1. 在维尼量子节点中生成订阅链接
2. 复制链接
3. 在手机客户端中添加订阅
4. 更新订阅获取所有节点

## 📖 详细配置教程

### Clash for Android配置

#### 1. 安装Clash for Android

1. 从GitHub或F-Droid下载APK
2. 允许安装未知来源应用
3. 安装并打开

#### 2. 导入配置

**方式A: 扫描二维码**

```
1. 打开Clash
2. 点击右上角 "+" 
3. 选择"扫描二维码"
4. 扫描电脑上的二维码
5. 自动导入完成
```

**方式B: URL导入**

```
1. 复制订阅链接
2. 打开Clash
3. 点击 "配置" → "新配置" → "URL"
4. 粘贴链接
5. 点击"下载"
6. 导入完成
```

**方式C: 手动添加单个节点**

```
1. 复制节点链接（ss://或vmess://开头）
2. 打开Clash
3. 点击 "配置" → "新配置"
4. 选择"从剪贴板导入"
5. 自动识别并添加
```

#### 3. 启动代理

```
1. 在主界面选择配置文件
2. 选择节点（点击"代理"标签）
3. 点击底部的启动按钮
4. 首次使用会提示创建VPN连接，点击"确定"
5. 连接成功后会显示"已连接"
```

#### 4. 测试连接

1. 打开浏览器
2. 访问 Google.com 或 Twitter.com
3. 能打开说明配置成功！

## 🔧 高级配置

### 分流规则设置（Clash）

让国内网站直连，国外网站走代理：

```
1. 打开Clash设置
2. 进入"配置"
3. 选择"规则"标签
4. 设置规则模式:
   - 规则模式: 国内直连，国外代理
   - 直连模式: 全部直连
   - 全局模式: 全部走代理
```

### 自动切换节点

```
1. 在代理标签中选择"自动选择"组
2. Clash会自动测速
3. 选择最快的节点
```

### 节点延迟测试

```
1. 长按节点
2. 选择"延迟测试"
3. 查看延迟数值
4. 延迟越低速度越快
```

## 💡 使用技巧

### 1. 省电模式

**问题**: 代理一直运行很耗电
**解决**: 
- 使用"仅代理特定应用"功能
- 不用时及时关闭代理
- 选择离你较近的节点

### 2. 节点选择

**推荐顺序**:
1. 地理位置近的节点（延迟低）
2. 负载低的节点（速度快）
3. 最近验证可用的节点

### 3. 应用分流

让某些应用走代理，其他直连：

```
Clash设置步骤:
1. 设置 → 网络 → 访问控制
2. 选择"仅代理"
3. 添加需要代理的应用
```

### 4. 订阅更新

定期更新订阅获取最新节点：

```
1. 打开订阅设置
2. 下拉刷新
3. 或设置自动更新（每天/每周）
```

## ❓ 常见问题

### Q1: 提示"无法连接到互联网"

**原因**: VPN权限未授予或节点失效

**解决**:
1. 检查VPN权限是否允许
2. 尝试切换其他节点
3. 更新订阅获取新节点
4. 重启手机

### Q2: 连接后还是打不开网站

**原因**: DNS污染或规则问题

**解决**:
1. 清除浏览器缓存
2. 尝试使用全局模式
3. 更换DNS设置（如1.1.1.1）
4. 重新启动代理

### Q3: 部分应用无法使用

**原因**: 应用检测到代理或未被分流

**解决**:
1. 添加应用到代理列表
2. 使用全局模式
3. 尝试不同的节点

### Q4: 速度很慢

**原因**: 节点负载高或距离远

**解决**:
1. 切换其他节点
2. 进行节点延迟测试
3. 选择离你近的服务器
4. 避开高峰时段

### Q5: 无法下载某些应用

**原因**: Google Play被墙

**解决**:
1. 确保代理已开启
2. 清除Play Store缓存
3. 使用全局模式
4. 或从其他来源下载（如APKPure）
