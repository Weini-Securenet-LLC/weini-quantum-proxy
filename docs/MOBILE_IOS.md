## 🛒 如何获取客户端

### 方式一：购买美区/港区Apple ID（推荐）

1. **购买现成账号**
   - 淘宝搜索"美区Apple ID"
   - 价格约10-30元
   - 只用于下载，不要登录iCloud

2. **自己注册**
   - 访问 appleid.apple.com
   - 选择美国/香港地区
   - 填写信息注册
   - 需要美国/香港地址（可搜索生成器）

3. **下载步骤**
   ```
   1. App Store退出当前账号
   2. 登录美区/港区账号
   3. 搜索并下载Shadowrocket
   4. 下载完成后退出
   5. 重新登录自己的账号
   ```

### 方式二：使用TestFlight版本（免费但功能有限）

部分应用提供TestFlight测试版本，可免费使用。

## 📖 详细配置教程

### Shadowrocket（小火箭）配置

#### 1. 首次启动设置

```
1. 打开Shadowrocket
2. 允许通知（可选）
3. 允许添加VPN配置
4. 输入密码或使用Face ID确认
```

#### 2. 导入节点

**方式A: 扫描二维码**

```
1. 首页点击右上角 "+" 
2. 选择"扫描二维码"
3. 扫描电脑上的节点二维码
4. 自动添加到列表
```

**方式B: 从剪贴板导入**

```
1. 复制节点链接（ss://或vmess://）
2. 打开Shadowrocket
3. 应用会自动检测剪贴板
4. 弹出提示，点击"确定"添加
```

**方式C: 添加订阅**

```
1. 首页点击右上角 "+"
2. 选择"Subscribe"（订阅）
3. URL: 粘贴订阅链接
4. 备注: 输入名称（如"维尼节点"）
5. 点击"完成"
6. 下拉刷新更新订阅
```

**方式D: 手动添加节点**

```
1. 点击右上角 "+"
2. 选择"类型"（SS/VMess/VLESS/Trojan）
3. 填写节点信息:
   - 地址: 服务器地址
   - 端口: 端口号
   - 密码/UUID: 认证信息
   - 加密方式: 根据节点选择
4. 保存
```

#### 3. 启动代理

```
1. 在首页选择一个节点（点击节点行）
2. 节点会显示黄色圆点（已选中）
3. 打开顶部的连接开关
4. 首次使用会提示"添加VPN配置"
5. 点击"Allow"（允许）
6. 输入密码或Face ID确认
7. 状态栏出现VPN图标表示连接成功
```

#### 4. 测试连接

```
1. 打开Safari浏览器
2. 访问 www.google.com 或 twitter.com
3. 能打开说明配置成功！
```

## 🔧 高级配置

### 分流规则（Shadowrocket）

让国内网站直连，国外网站走代理：

```
1. 设置 → 配置
2. 点击"默认配置"
3. 编辑规则:
   - GEOIP,CN,DIRECT （中国IP直连）
   - FINAL,PROXY （其他走代理）
4. 保存
```

### 场景模式

不同场景使用不同配置：

```
1. 设置 → 场景
2. 创建新场景:
   - 场景1: 全局代理
   - 场景2: 规则分流
   - 场景3: 全部直连
3. 根据需要快速切换
```

### 节点延迟测试

```
1. 在节点列表
2. 点击"连通性测试"
3. 查看各节点延迟
4. 选择延迟最低的节点
```

### 自动测速选择

```
1. 设置 → 延迟测试方法
2. 选择"ICMP"（更准确）
3. 开启"自动测试"
4. 自动选择最快节点
```

**问题**: VPN一直运行很耗电

**解决**:
- 使用"按需连接"功能
- 不用时关闭VPN
- 选择离你近的节点

## ❓ 常见问题

### Q1: 提示"无法连接"或"请求超时"

**原因**: 节点失效或网络问题

**解决**:
1. 切换其他节点
2. 更新订阅获取新节点
3. 检查本地网络连接
4. 重启应用和手机

### Q2: 连接后还是打不开网站

**原因**: DNS污染或规则配置问题

**解决**:
1. 切换全局代理模式
2. 更改DNS设置（如1.1.1.1）
3. 清除Safari缓存
4. 重新选择节点

### Q3: App Store无法下载应用

**原因**: DNS缓存或账号地区问题

**解决**:
1. 断开VPN重试
2. 切换使用规则模式
3. 清除App Store缓存
4. 确认Apple ID地区设置

### Q4: 微信等国内应用无法使用

**原因**: 全局代理导致国内应用异常

**解决**:
1. 使用规则分流模式
2. 添加微信到直连规则
3. 或临时关闭代理

### Q5: 速度很慢或经常断开

**原因**: 节点负载高或不稳定

**解决**:
1. 进行节点测速
2. 选择延迟低的节点
3. 避开使用高峰期
4. 更新节点列表

### Q6: 付费后无法下载Shadowrocket

**原因**: 地区限制或账号问题

**解决**:
1. 确认使用美区/港区账号
2. 检查Apple ID余额
3. 尝试更换美区账号
4. 联系App Store支持

## 🔒 安全建议

### 1. Apple ID安全

- ✅ 只在App Store使用美区ID
- ❌ 不要在设置-iCloud中登录美区ID
- ⚠️ 不要在美区ID中存储个人信息
- 🔐 定期更换美区ID密码

### 2. 节点安全

- ✅ 使用维尼量子节点验证的节点
- ✅ 定期更新节点列表
- ⚠️ 不要使用来路不明的节点
- 🔐 避免在代理下输入银行密码

### 3. 应用安全

- ✅ 从官方App Store下载
- ❌ 不要安装来历不明的企业签名版本
- ⚠️ 定期检查应用权限
- 🔐 使用Face ID保护应用

### 4. 隐私保护

- 🔐 使用HTTPS网站
- 🔐 不要在公共WiFi下使用敏感应用
- 🔐 定期清理浏览器缓存
