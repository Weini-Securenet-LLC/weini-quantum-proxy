## 简体中文

### ✨ 项目特色

维尼量子节点是一个**完整的开源生态系统**，不仅包括代理应用程序，还包括AI Agent可用的智能工具。

#### 🖥️ 桌面应用程序
- ✅ **多协议支持**: SS / VMess / VLESS / Trojan
- ✅ **智能节点发现**: 自动从多个来源聚合可用节点
- ✅ **真实验证**: TCP可达性 + 协议验证
- ✅ **一键代理**: 简单易用，无需复杂配置
- ✅ **实时监控**: 流量、速度、连接状态实时显示
- ✅ **跨平台**: Windows / macOS / Linux 全支持

#### 🤖 AI Agent Skills（智能工具）
- ✅ **节点抓取Skill**: 从GitHub聚合免费节点
- ✅ **节点验证Skill**: 自动验证节点可用性
- ✅ **自动化流程**: 定期更新节点列表
- ✅ **开源可复用**: 任何人都可以使用这些技能
- 🎯 **赋能他人**: 让更多人用AI工具实现通信自由

#### 📱 移动设备支持
- 📖 **安卓设备指南**: 详细的配置教程
- 📖 **iOS设备指南**: 完整的使用说明
- 🔄 **节点分享**: 轻松分享节点给朋友
- 💾 **本地配置**: 无需注册，开箱即用

#### 🛡️ 隐私和安全
- 🔒 开源透明，代码可审查
- 🔐 加密传输，保护隐私
- 🚫 不记录用户数据
- 🆓 完全免费，无广告无追踪

### 🎬 快速开始

#### 方式1: 桌面应用（推荐新手）

1. 前往 [Releases](https://github.com/Weini-Securenet-LLC/weini-quantum-proxy/releases) 下载适合你系统的版本
2. 解压并运行程序
3. 选择一个节点
4. 点击"启动全局代理"
5. 享受自由的互联网 🎉

#### 方式2: 移动设备

- **安卓用户**: 查看 [安卓设备配置指南](docs/MOBILE_ANDROID.md)
- **iOS用户**: 查看 [iOS设备配置指南](docs/MOBILE_IOS.md)

#### 方式3: 使用AI工具自动化

如果你使用AI编程工具（如Cursor、GitHub Copilot等），可以使用我们的Skills：

- **抓取节点**: `skills/research/weini-proxy/`
- **验证节点**: `skills/research/proxy-node-studio-list-refresh/`

查看 [Skill开发指南](docs/SKILL_DEVELOPMENT.md) 了解如何使用和扩展。

### 🤖 为什么开源AI Skills很重要？

**让技术为自由服务**

我们相信，不仅代理工具应该开源，连**获取和维护节点的方法**也应该开源。通过开源AI Agent Skills：

1. **赋能个人**: 任何人都可以用AI工具为自己获取节点
2. **帮助他人**: 用AI自动化流程，为社区提供节点
3. **知识传播**: 分享方法比分享节点更有价值
4. **可持续性**: 去中心化的节点获取，不依赖单一来源

**使用场景**:
- 🎯 个人用户用AI助手自动获取最新节点
- 🤝 技术志愿者用AI批量验证和分享节点
- 🔄 开发者集成Skills到自己的工具中
- 📚 教育和培训如何使用AI实现自由

### 📖 为什么需要维尼量子节点？

在世界的许多角落，人们无法自由访问互联网：

- 🚫 **中国**: 长城防火墙封锁Google、Twitter、Facebook等
- 🚫 **伊朗**: 严格的网络审查和社交媒体封锁  
- 🚫 **俄罗斯**: 封锁反对派网站和独立媒体
- 🚫 **土耳其**: 限制社交媒体和新闻网站
- 🚫 **缅甸**: 军政府封锁互联网

**维尼量子节点帮助你**:
- 🌐 访问被封锁的网站和服务
- 📰 获取不受审查的新闻资讯
- 💬 与世界自由交流
- 🎓 访问教育和学术资源
- 🆘 在紧急情况下保持连接
- 📱 在手机和电脑上都能使用
