## ✨ Project Features

Weini Quantum Proxy is a **complete open-source ecosystem**, including not only proxy applications but also AI Agent skills.

### 🖥️ Desktop Application
- ✅ **Multi-Protocol Support**: SS / VMess / VLESS / Trojan
- ✅ **Smart Node Discovery**: Automatically aggregates available nodes from multiple sources
- ✅ **Real Validation**: TCP reachability + protocol validation
- ✅ **One-Click Proxy**: Simple to use, no complex configuration needed
- ✅ **Real-Time Monitoring**: Live display of traffic, speed, and connection status
- ✅ **Cross-Platform**: Full support for Windows / macOS / Linux

### 🤖 AI Agent Skills
- ✅ **Node Crawling Skill**: Aggregate free nodes from GitHub
- ✅ **Node Validation Skill**: Automatically validate node availability
- ✅ **Automation**: Regularly update node lists
- ✅ **Open & Reusable**: Anyone can use these skills
- 🎯 **Empower Others**: Let more people use AI tools for communication freedom

### 📱 Mobile Device Support
- 📖 **Android Guide**: Detailed configuration tutorial
- 📖 **iOS Guide**: Complete usage instructions
- 🔄 **Node Sharing**: Easily share nodes with friends
- 🌐 **Cross-Device Sync**: One account, multiple devices

### 🛡️ Privacy & Security
- 🔒 Open-source and transparent, code can be audited
- 🔐 Encrypted transmission protects privacy
- 🚫 No user data logging
- 🆓 Completely free, no ads, no tracking

## 🎬 Quick Start

### Method 1: Desktop Application (Recommended for Beginners)

1. Go to [Releases](https://github.com/Weini-Securenet-LLC/weini-quantum-proxy/releases) and download the version for your system
2. Extract and run the program
3. Select a node
4. Click "Start Global Proxy"
5. Enjoy the free internet 🎉

### Method 2: Mobile Devices

- **Android Users**: See [Android Configuration Guide](../docs/MOBILE_ANDROID.md)
- **iOS Users**: See [iOS Configuration Guide](../docs/MOBILE_IOS.md)

### Method 3: Use AI Tools for Automation

If you use AI coding tools (like Cursor, GitHub Copilot, etc.), you can use our Skills:

- **Crawl Nodes**: `skills/research/weini-proxy/`
- **Validate Nodes**: `skills/research/proxy-node-studio-list-refresh/`

See [Skill Development Guide](../docs/SKILL_DEVELOPMENT.md) to learn how to use and extend.

## 🏗️ Technical Architecture

```
Weini Quantum Proxy
├── 🖥️  Desktop Application (Wails + Go)
│   └── Clean and friendly user interface
├── 🔍 Node Discovery System
│   ├── GitHub subscription aggregation
│   ├── Multi-format parsing (Clash/URI)
│   └── Smart deduplication and filtering
├── ✅ Node Validation System
│   ├── TCP reachability detection
│   ├── Real protocol validation
│   └── Geographic information tagging
└── 🚀 Proxy Runtime
    ├── sing-box core
    ├── One-click system proxy configuration
    └── Real-time traffic monitoring
```
